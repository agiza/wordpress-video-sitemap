Provides a XML Video Sitemap for quicker indexing of your youtube videos. 

== Description ==
WordPress Video Sitemaps is an addon module to Yoast's WordPress SEO plugin. 

= XML Video Sitemap =
This addon extends the sitemap_index.xml file by adding an entry for the video sitemap. This implies that the search engines will automatically retrieve the video sitemap without you adding it via webmaster tools.

== Installation ==

1. Upload the `wordress-video-sitemap` folder to the `/wp-content/plugins/` directory
2. Activate the WordPress Video Sitemap plugin through the 'Plugins' menu in WordPress
